# 移动易 —— 让移动应用开发更容易！

## 启动应用步骤：

 - git clone https://github.com/sectong/yidongyi ;
 - import to Eclipse；
 - run

```
	
	cd yidongyi
	mvn spring-boot:run
	...
	...
	log出现：Tomcat started on port(s): 8080 (http)，证明启动成功

```

## 管理系统
